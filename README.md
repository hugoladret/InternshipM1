Les notebooks requièrent les packages suivants :
* numpy
* matplotlib
* pytorch

### Pour la production de saccades 
* [LogGabor](https://pypi.org/project/LogGabor/)
* [SparseEdges](https://pypi.org/project/SparseEdges/20171205/#description)
